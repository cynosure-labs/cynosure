# Hello Triangle

Renders a spinning triangle to a [winit](https://github.com/rust-windowing/winit) window.

## To Run

> Root directory of the project is assumed.

```
cargo run --bin hello-triangle
```
or
```
cargo run-wasm --bin hello-triangle
```
